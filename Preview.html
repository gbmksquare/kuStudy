Description
kuStudy provides live reading room seats information of Korea University.

Features:
• Total seats in a library
• Used & available seats in a library
• Total seats in a reading room
• Used & available seats in a reading room

Supports:
• iOS app
• Today widget
• Apple Watch app
• Apple Watch Glance

Note: Depending on University's server status, information cant' be shown.

kuStudy는 고려대학교 안암 캠퍼스 내의 열람실 좌석 현황을 실시간으로 제공합니다.

기능:
• 도서관 전체 좌석 수
• 도서관 사용 및 잔여 좌석 수
• 열람실 전체 좌석 수
• 열람실 사용 및 잔여 좌석 수

지원:
• iOS 앱
• 오늘 위젯
• Apple Watch 앱
• Apple Watch 한눈에 보기

주의: 학교 서버 상태에 따라 열람실 정보를 표시하지 못할 수 있습니다.
